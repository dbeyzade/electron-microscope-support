# Electron Microscope Support

Welcome to the official support page for Electron Microscope (macOS).

## Contact
- Open a support issue: https://github.com/dbeyzade/electron-microscope-support/issues

## Common Help
- App startup issue: Ensure macOS version is up to date and reinstall the latest app build.
- Data input issue: Accepted formats are Hex, Binary, and Decimal byte streams.
- Analysis issue: Try smaller datasets first, then increase data size gradually.

## Privacy
Electron Microscope processes analysis data locally on your Mac.
