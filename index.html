Common Questions

The app does not start.
Update macOS, reinstall the latest build, and relaunch from Applications.

My data is not parsed correctly.
Supported formats are Hex, Binary, and Decimal byte streams. Remove unsupported separators and text labels.

Analysis output looks empty.
Test with a smaller dataset first, then scale up gradually. Extremely sparse input can produce low-contrast output.

Is data uploaded to a server?
No. Analysis data is processed locally on your Mac.
