Quick Troubleshooting Flow

  1. Confirm you are using a supported macOS version and the latest app build.
  2. Validate your input format before import: Hex, Binary, or Decimal byte stream.
  3. Run a smaller dataset first to verify parsing and visualization path.
  4. Apply one analysis at a time to isolate the source of unexpected output.
  5. If the issue persists, email logs, screenshot, and sample input to support.

Data Format Guide

Examples of accepted inputs:

  • Hex: 0A 1F 2B 3C
  • Binary: 00001010 00011111
  • Decimal: 10 31 43 60

Tip: Remove labels and non-byte symbols before paste/import.

Frequently Asked Questions

The app does not open after installation.

Reinstall the latest build, then launch from the Applications folder. If Gatekeeper blocks startup, open System Settings and allow the app, then retry.

My image looks blank after loading data.

This usually means data density is low or format detection is wrong. Try explicit format selection and run a smaller known-good sample first.

Particle analysis count seems inaccurate.

The current method is threshold-based. For better results, preprocess with histogram equalization and test multiple threshold ranges.

Does the app send my data to an external server?

No. Analysis runs locally on your Mac device. Data is not uploaded by default.
